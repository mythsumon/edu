## 시스템 개요

**교육 관리 시스템**은 교육 프로그램, 교육기관, 강사, 교육 과정을 통합 관리하는 웹 애플리케이션입니다.

### 주요 기능
- 📊 대시보드: 전체 프로그램 현황 및 권역별 통계
- 📚 교육 운영: 교육 과정 등록 및 관리
- 👨‍🏫 강사 배정: 강사 신청, 배정, 출강 확정 관리
- 🏢 기준정보 관리: 교육기관, 프로그램, 강사 정보 관리
- ⚙️ 시스템 관리: 설정 및 사용자 관리

---

## 주요 사용자 역할

### 관리자 (Admin)
- 모든 기능에 대한 전체 접근 권한
- 기준정보 관리 및 시스템 설정 관리

### 운영자 (Operator)
- 교육 운영 및 강사 배정 관리
- 교육 과정 등록 및 수정

## 전체 사용자 플로우

```
[로그인]
    ↓
[대시보드] - 전체 프로그램 현황 확인
    ↓
    ├─→ [교육 운영] - 교육 관리
    │       ├─→ 목록 조회
    │       ├─→ 교육 등록
    │       └─→ 교육 수정/삭제
    │
    ├─→ [강사 배정]
    │       ├─→ 강사 신청 관리
    │       ├─→ 강사 배정 관리
    │       └─→ 출강 확정 관리
    │
    ├─→ [기준정보 관리]
    │       ├─→ 교육기관 관리
    │       │       ├─→ 목록 조회
    │       │       ├─→ 기관 등록
    │       │       └─→ 기관 수정/삭제
    │       │
    │       ├─→ 프로그램 관리
    │       │       ├─→ 목록 조회
    │       │       ├─→ 프로그램 등록
    │       │       └─→ 프로그램 수정/삭제
    │       │
    │       └─→ 강사 관리
    │               ├─→ 목록 조회
    │               ├─→ 강사 등록
    │               └─→ 강사 수정/삭제
    │
    └─→ [시스템 관리]
            └─→ 설정 및 사용자 관리
```

## 상세 기능별 플로우

### 1. 대시보드 (전체 프로그램 현황)

```
[대시보드 접근]
    ↓
[주요 지표 확인]
    - 전체 프로그램 수
    - 진행 중인 교육 수
    - 완료된 교육 수
    - 등록된 강사 수
    ↓
[권역별 교육 진행 현황]
    - 6개 권역 카드 표시
    - 권역 클릭 → 상세 정보 표시
    ↓
[권역 지도]
    - 지도에서 권역 선택 가능
    - 선택된 권역 하이라이트
    ↓
[특수 항목 진행률]
    - 3개 특수 항목 카드 표시
    - 항목 클릭 → 상세 정보 표시
    ↓
[전체 프로그램 리스트]
    - 프로그램 목록 조회
    - 검색 및 필터링
    - 페이지네이션
```

### 2. 교육 관리 플로우

```
[교육 관리 메뉴 클릭]
    ↓
[교육 목록 페이지]
    ├─→ [검색/필터]
    │       - 교육명 검색
    │       - 상태 필터
    │       - 날짜 범위 필터
    │
    ├─→ [교육 등록]
    │       ↓
    │   [교육 기본 정보 입력]
    │       - 프로그램 선택
    │       - 교육기관 선택
    │       - 교육명 입력
    │       - 설명 입력
    │       - 시작일/종료일 선택
    │       - 상태 선택
    │       ↓
    │   [학급 정보 입력]
    │       - 학년, 반, 학생 수 입력
    │       ↓
    │   [수업 정보 입력]
    │       - 수업 개수 입력
    │       - 각 수업별 일자, 시간, 강사 수 입력
    │       ↓
    │   [저장] → 목록으로 돌아가기
    │
    ├─→ [교육 수정]
    │       - 목록에서 교육 클릭
    │       - 정보 수정 후 저장
    │
    └─→ [교육 삭제]
            - 체크박스로 선택
            - 삭제 버튼 클릭
            - 확인 후 삭제
```

## 페이지별 상세 플로우

### 대시보드 페이지 (`/`)

**초기 상태:**
- 권역별 교육 진행 현황 (6개 카드)
- 권역 지도
- 특수 항목 진행률 (3개 카드)

**상호작용:**
1. 권역 카드 클릭
   - 선택된 권역 하이라이트
   - 결과 패널에 상세 정보 표시
   - "뒤로" 버튼 표시

2. 권역 지도에서 권역 클릭
   - 해당 권역 하이라이트
   - 결과 패널에 상세 정보 표시

3. 특수 항목 카드 클릭
   - 선택된 항목 하이라이트
   - 결과 패널에 상세 정보 표시

4. "뒤로" 버튼 클릭
   - 초기 상태로 복귀
   - 검색 패널 표시

### 교육 관리 페이지 (`/education`)

**목록 뷰:**
- 테이블: 교육ID, 교육명, 프로그램, 교육기관, 상태, 교육기간
- 체크박스 선택 → 삭제 버튼 표시
- "교육 등록" 버튼 → 등록 뷰로 전환

**등록/수정 뷰:**
- 왼쪽: 섹션 네비게이션 (교육 기본 정보, 학급 정보, 수업 정보)
- 오른쪽: 폼 입력 영역
- "저장" 버튼 → 목록 뷰로 복귀
- "취소" 버튼 → 목록 뷰로 복귀

### 교육기관 관리 페이지 (`/institution`)

**목록 뷰:**
- 필터: 권역, 기관명, 담당자 검색
- 테이블: 권역, 기관ID, 기관명, 주소, 전화번호, 담당자
- 체크박스 선택 → 삭제 버튼 표시
- "기관 등록" 버튼 → 등록 뷰로 전환

**등록/수정 뷰:**
- 왼쪽: 섹션 네비게이션 (교육기관 정보, 담당자 정보)
- 오른쪽: 폼 입력 영역
- "저장" 버튼 → 목록 뷰로 복귀
- "취소" 버튼 → 목록 뷰로 복귀

### 프로그램 관리 페이지 (`/program`)

**목록 뷰:**
- 필터: 프로그램명 검색, 상태 선택
- 테이블: 상태, 프로그램ID, 프로그램명, 비고, 등록일시
- 체크박스 선택 → 삭제 버튼 표시
- "프로그램 등록" 버튼 → 등록 뷰로 전환

**등록 뷰:**
- 왼쪽: 섹션 네비게이션 (프로그램 정보)
- 오른쪽: 폼 입력 영역
- "등록하기" 버튼 → 목록 뷰로 복귀
- "뒤로" 버튼 → 목록 뷰로 복귀

### 강사 관리 페이지 (`/instructor`)

**목록 뷰:**
- 필터: 강사명/계정 검색, 지역 선택, 강사 구분 선택, 초기화
- 테이블: 강사ID, 강사명, 계정, 소속, 지역, 강사 구분
- 체크박스 선택 → 삭제 버튼 표시
- "강사 등록" 버튼 → 등록 뷰로 전환

**등록/수정 뷰:**
- 왼쪽: 섹션 네비게이션 (기본 정보, 소속/지역 정보, 운영 정보)
- 오른쪽: 폼 입력 영역
- "등록하기/저장하기" 버튼 → 목록 뷰로 복귀
- "뒤로" 버튼 → 목록 뷰로 복귀

## 주요 사용자 시나리오

### 시나리오 1: 새로운 교육 과정 등록

1. **대시보드 접근** → 전체 프로그램 현황 확인
2. **교육 운영** → 교육 관리 메뉴 클릭
3. **교육 등록** 버튼 클릭
4. **교육 기본 정보** 입력
   - 프로그램 선택 (기준정보에서 등록된 프로그램)
   - 교육기관 선택 (기준정보에서 등록된 기관)
   - 교육명, 설명, 기간 입력
5. **학급 정보** 입력
   - 학년, 반, 학생 수 입력
6. **수업 정보** 입력
   - 수업 개수 입력
   - 각 수업별 일정 및 강사 수 입력
7. **저장** → 교육 목록에 추가됨

### 시나리오 2: 강사 등록 및 배정

1. **기준정보 관리** → 강사 관리 메뉴 클릭
2. **강사 등록** 버튼 클릭
3. **기본 정보** 입력
   - 사용자명, 비밀번호, 강사명, 연락처 등
4. **소속/지역 정보** 입력
   - 소속, 지역 선택
5. **운영 정보** 입력
   - 상태, 강사 구분 선택
6. **등록하기** → 강사 목록에 추가됨
7. **강사 배정** 메뉴로 이동하여 배정 관리

### 시나리오 3: 권역별 교육 현황 조회

1. **대시보드** 접근
2. **권역별 교육 진행 현황** 섹션에서 권역 카드 클릭
   또는 **권역 지도**에서 권역 클릭
3. **결과 패널**에 상세 정보 표시
   - 전체 교육 진행률
   - 요약 테이블
   - 프로그램별 상세 정보
4. **특수 항목** 카드 클릭하여 특수 항목별 상세 조회
5. **"뒤로"** 버튼으로 초기 상태 복귀

## UI/UX 특징

### 공통 디자인 시스템
- **Input 필드**: 높이 `h-11`, 둥근 모서리 `rounded-xl`
- **버튼**: 높이 `h-11`, 패딩 `px-6`, 둥근 모서리 `rounded-xl`
- **Primary 버튼**: 파란색 배경 (`bg-blue-600`)
- **Secondary 버튼**: 회색 테두리 (`border-gray-300`)
- **Danger 버튼**: 빨간색 배경 (Ant Design danger prop)

### 레이아웃 패턴
- **목록 뷰**: 필터 카드 + 테이블 카드
- **등록/수정 뷰**: 왼쪽 섹션 네비게이션 + 오른쪽 폼 영역
- **반응형**: 모바일 1열, 데스크톱 2열 그리드

### 네비게이션
- **사이드바**: 항상 열려있음, 활성 메뉴 하이라이트
- **브레드크럼**: 현재 위치 표시
- **뒤로 버튼**: 폼 뷰에서 목록 뷰로 복귀
